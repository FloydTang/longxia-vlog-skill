# 出片质量检查清单

## 必过项

- [ ] 开头 3 秒能看懂这是一次学习 / 活动 / 成长记录。
- [ ] 视频不是纯模板堆砌，至少 3 个真实素材进入主体。
- [ ] 字幕没有贴底，没有被平台 UI 或水印遮挡。
- [ ] 人脸、产品、关键物体没有被标题挡住。
- [ ] TTS 清楚，BGM 没有盖过人声。
- [ ] 正式交付未使用系统 TTS 占位音。
- [ ] 片头、片尾和水印品牌一致。
- [ ] 片头和片尾优先使用真实场景照片，而不是随机 AI 背景。
- [ ] 没有 AI 生成水印、乱码字、错别字。
- [ ] 没有黑屏、空白、素材路径错误。
- [ ] 没有敏感客户信息、付款信息、后台数据。
- [ ] 片尾 CTA 不超过 6 秒，不喧宾夺主。

## 建议项

- [ ] 每 5-8 秒至少有一次画面变化。
- [ ] 主字幕每屏不超过两行。
- [ ] 旁白不是朗读流水账，而是有清晰情绪线。
- [ ] 结尾能自然引向下一次课程、社媒作业或作战台任务。

## 返工判断

任一情况出现，必须返工：

- BGM 明显难听或刺耳。
- 画面和旁白说的不是一回事。
- 片头/片尾像随机 AI 图，和课程品牌无关。
- 片头/片尾出现蓝紫科技隧道、抽象方块或“AI生成”水印。
- 学员真实素材太少，成片主要靠模板撑时长。
- 视频看起来像技术 demo，不像可以发给学员的作品。
