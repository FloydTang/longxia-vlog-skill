# 素材与分镜匹配清单

## 素材分类

| 类型 | 判断 | 默认用法 |
| --- | --- | --- |
| 人物近景 | 表情清楚、主体明确 | 放在收获、行动、金句段落 |
| 合影 | 人多但信息明确 | 放在开头后或结尾前 |
| 课堂/活动现场 | 能看出场景 | 放在交代背景 |
| 物料/笔记/屏幕 | 有学习痕迹 | 放在“学到了什么” |
| 风景/路上 | 氛围素材 | 做过渡，不承载核心信息 |
| 横屏视频 | 可裁切、有动作 | 用短段落，避免裁掉主体 |
| 截图 | 信息明确且不敏感 | 少用，优先转成局部画面 |

## 匹配规则

1. 讲“从哪里来”时，用路上、场地、第一张人物图。
2. 讲“学到了什么”时，用课堂、笔记、现场互动。
3. 讲“准备做什么”时，用人物近景、电脑、行动相关素材。
4. 讲“给自己一句话”时，用最有情绪的一张照片。
5. 片尾不要突然放无关素材。

## 不匹配时的处理

- 文案讲课堂，画面却是风景：改文案或换素材。
- 文案讲行动，画面只有合影：补一句“从合影开始，下一步是行动”。
- 素材全是风景：视频定位改为“学习路上的记录”，不要硬讲课程成果。
- 画面识别不确定：不要编造细节，用更宽泛的表达。

## 最低素材要求

- 35 秒以内：至少 3 个有效素材。
- 35-60 秒：至少 5 个有效素材。
- 60 秒以上：至少 7 个有效素材或 2 段可用视频。

低于最低素材要求时，先让学员补图，不要硬生成。
